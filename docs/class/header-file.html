Header File

Δημιουργία του Header File

  1. #ifndef (If Not Defined)
    • Ελέγχει αν η μακροεντολή έχει ήδη οριστεί.
    • Αν δεν είναι ορισμένη, προχωράει στην εκτέλεση του επόμενου κώδικα.
  2. #define
    • Ορίζει τη μακροεντολή ώστε την επόμενη φορά που ο compiler συναντήσει το ίδιο header file, να παραλείψει τον κώδικα που βρίσκεται μεταξύ του #ifndef και του #endif.
  3. #endif
    • Κλείνει τον conditional block, δηλαδή τον κώδικα που θα εκτελεστεί μόνο αν η μακροεντολή δεν ήταν ήδη ορισμένη.

Compiling

g++ main.cpp my_functions.cpp -o my_program
