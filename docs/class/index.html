C++ Class

Δημιουργήστε μια class

Για να δημιουργήσετε μια class, χρησιμοποιήστε τη λέξη-κλειδί class:

class MyClass {       // The class
public:             // Access specifier
int myNum;        // Attribute (int variable)
string myString;  // Attribute (string variable)
};

Η λέξη-κλειδί Public είναι ένας προσδιοριστής πρόσβασης. Οι προσδιοριστές πρόσβασης καθορίζουν τον τρόπο πρόσβασης στα μέλη (χαρακτηριστικά και μέθοδοι) μιας κλάσης.
Στην C++, υπάρχουν τρεις προσδιοριστές πρόσβασης:

  1. Public : ολα τα στοιχεια είναι προσβάσιμα εκτός της class
  2. Private - δεν είναι δυνατή η πρόσβαση (ή η προβολή) των μελών εκτός της class
  3. Protected - τα μέλη δεν είναι προσβάσιμα εκτός της class, ωστόσο, μπορούν να προσπελαστούν σε κλάσεις κληρονομιάς.

Κατασκευαστές

#include <iostream.h>
class CRectangle {
int width, height;
public:
CRectangle (int,int);
int area (void) {return (width*height);}
};

CRectangle::CRectangle (int a, int b) {
width = a;
height = b;
}

int main () {
CRectangle rect (3,4);
CRectangle rectb (5,6);
cout <<  "rect area: " << rect.area() << endl;
cout << "rectb area: " << rectb.area() << endl;
}

Καταστροφείς

Ο καταστροφέας (destructor) επιτελεί την αντίστροφη λειτουργία από αυτή του κατασκευαστή. Καλείται αυτόματα όταν ένα αντικείμενο εξέρχεται από τη μνήμη, είτε διότι σταμάτησε να υφίσταται λόγος ύπαρξής του (π.χ. Ήταν δηλωμένο ως τοπικό αντικείμενο σε μια συνάρτηση της οποίας η εκτέλεση ολοκληρώθηκε), είτε επειδή είναι ένα δυναμικά ορισμένο αντικείμενο το οποίο αποδεσμεύεται μέσω του τελεστή delete.

Ο καταστροφέας δηλώνεται ως μια συνάρτηση-μέλος, η οποία έχει ακριβώς το ίδιο όνομα με την κλάση με μία τίλντα (~) μπροστά. 

#include <iostream.h>

class CRectangle {
int *width, *height;
public:
CRectangle (int,int);
~CRectangle ();
int area (void) {return (*width * *height);}
};

CRectangle::CRectangle (int a, int b) {
width = new int;
height = new int;
*width = a;
*height = b;
}
CRectangle::~CRectangle () {
delete width;
delete height;
}

int main () {
CRectangle rect (3,4), rectb (5,6);
cout <<  "rect area: " << rect.area() << endl;
cout << "rectb area: " << rectb.area() << endl;
return 0;
}

Virtual

  1. Η λέξη-κλειδί virtual επιτρέπει σε μέλος παράγωγης κλάσης που έχει το ίδιο όνομα με ένα μέλος της βασικής κλάσης, να καλείται από τα σχετικά αντικείμενα
  2. Ο βασικός λόγος χρήσης μιας ιδεατής συνάρτησης είναι η δυνατότητα διαφοροποίησης της λειτουργικότητας της συνάρτησης στις παράγωγες κλάσεις

Αφηρημένες βασικές κλάσεις

  1. Οι αφηρημένες (abstract) βασικές κλάσεις ορίζονται ως πρωτότυπα για τις παράγωγες κλάσεις τους. Δεν μπορούμε να ορίσουμε αντικείμενα για τις κλάσεις αυτές, παρά μόνο δείκτες σε αυτές.
  2. Μια αφηρημένη κλάση έχει τουλάχιστον μια pure virtual συνάρτηση, η οποία δεν ορίζει κώδικα υλοποίησης

Καθαρά Εικονικές Μέθοδοι (Pure Virtual Methods)
Η κύρια ιδιότητα μιας αφηρημένης κλάσης είναι ότι περιέχει τουλάχιστον μία καθαρά εικονική μέθοδο. Μια καθαρά εικονική μέθοδος δηλώνεται με τη μορφή:

virtual void methodName() = 0;

Η δήλωση αυτή σημαίνει ότι η κλάση δεν παρέχει υλοποίηση για τη συγκεκριμένη μέθοδο και οι παράγωγες κλάσεις πρέπει να την υλοποιήσουν.
