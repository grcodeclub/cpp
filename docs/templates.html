Templates συναρτήσεων

Η κλήση μιας template συνάρτησης γίνεται με τον εξής τρόπο:

function <pattern> (parameters);

Παράδειγμα

Δημιουργία Προτύπου Συνάρτησης:

  1. template <class T>: Δημιουργούμε ένα πρότυπο που ονομάζεται T. Το class εδώ δεν σημαίνει ότι το T πρέπει να είναι τάξη, μπορεί να είναι οποιοσδήποτε τύπος δεδομένων.
  2. T GetMax (T a, T b): Ορίζουμε μια συνάρτηση GetMax που δέχεται δύο παραμέτρους τύπου T και επιστρέφει τον τύπο T.

#include <iostream> // Χρησιμοποιούμε iostream για την είσοδο και έξοδο

// Δημιουργία προτύπου για την εύρεση του μέγιστου αριθμού
template <class T>
T GetMax (T a, T b) {
return (a > b) ? a : b;
}

int main () {
int i = 5, j = 6, k;
long l = 10, m = 5, n;

// Χρήση της προτυποποιημένης συνάρτησης GetMax για τύπο int
k = GetMax(i, j); // Ή k = GetMax(i, j); 
std::cout << "Το μεγαλύτερο μεταξύ " << i << " και " << j << " είναι " << k << std::endl;

// Χρήση της προτυποποιημένης συνάρτησης GetMax για τύπο long
n = GetMax(l, m); // Ή n = GetMax(l, m); 
std::cout << "Το μεγαλύτερο μεταξύ " << l << " και " << m << " είναι " << n << std::endl;

return 0;
}
